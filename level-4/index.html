Level 4 — Integration

Thinking shift: From two separate systems to one coherent product.

Knowledge domain: Full-stack integration, CORS, error handling, environment configuration, Twelve-Factor App, integration testing.

Prerequisite ceiling: Levels 1–3 complete.

Deliverable: A running full-stack application with the Django backend and React frontend communicating correctly across environments, with integration tests and environment parity.
