# Level 5 — Craft

**Thinking shift:** From building systems to building tools that build systems.

**Knowledge domain:** Code generation theory, Node.js CLI tools, template
engines, registry patching, generator testing.

**Prerequisite ceiling:** Levels 1–4 complete. Regular expressions (T2.8).

**Deliverable:** A working `stratum make:system <name>` CLI generator
that scaffolds both the Django backend and React frontend simultaneously,
with full test coverage.
