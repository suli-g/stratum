# Level 3 — Frontend

**Thinking shift:** From displaying data to managing state.

**Knowledge domain:** React hooks, state architecture, custom hooks,
component testing, Vite configuration.

HTML & CSS basics (T2.5).

**Deliverable:** A fully tested React frontend with a principled state
architecture using hooks and the Stratum store pattern.
