Stratum: Full-Stack Development from First Principles

Welcome to Stratum — a course that builds a full-stack React + Django framework with a CLI code generator from scratch, understanding every decision along the way.

Important

Before opening Lesson 1, read the Before You Begin document. It lists the knowledge this course assumes and links to resources for any gaps.

Course Structure

Stratum is organised into five levels, each representing a distinct thinking shift.

Level

Name

Thinking shift

1

Foundations

From coding to designing

2

Backend

From making it work to making it correct

3

Frontend

From displaying data to managing state

4

Integration

From two separate systems to one coherent product

5

Craft

From building systems to building tools that build systems
